Installation#

In order to work with Deepchecks Monitoring, you need to:

  1. Install with pip the deepchecks-client python package, which installs the SDK for interacting with the app and the Deepchecks Testing package.

  2. Log in to the Deepchecks Monitoring app and create an organization or join an existing organization

  3. Obtain an API key from the app

Install with pip#

deepchecks-client is a python package that installs the SDK for interacting with the monitoring app and the Deepchecks Testing package, which can be useful when exploring existing issues (e.g. when using the “download” feature to further research the issue in your local environment)

It can be installed using pip:

pip install deepchecks-client --upgrade

Create or join an organization#

Following an invite mail (sign up to our invite system at the Deepchecks Website, or contact us at info@deepchecks.com), you’ll be able to join a new organization. Then, head over to the deepchecks host URL (e.g. https://app.deepchecks.com) to start exploring.

Obtain your API key#

To start working with deepchecks, you must first create a DeepchecksClient object. To do that, you will need go generate a personal API token using the application’s dashboard:


Note: save your API key as you will be able to view it only once. We recommend seting an environment variable named DEEPCHECKS_API_TOKEN by running the following in your terminal:

export DEEPCHECKS_API_TOKEN=replace-this-string-with-your-api-token

If you need a new API key, the “regenerate” button will create and reveal a new key. This will invalidate the previously generated key. Then you’ll be able to create a DeepchecksClient and start interacting with the system.

You can use the following code snippet to instantiate it and start sending data to the system.

>>> import os
>>> from deepchecks_client import DeepchecksClient
>>> # it is recommended to store the token in an enviroment variable for security reasons.
>>> # alternatively (not recommended) you can replace the os.getenv function with the value of the token.
>>> host = os.getenv('DEEPCHECKS_API_HOST')
>>> token = os.getenv('DEEPCHECKS_API_TOKEN')
>>> dc_client = DeepchecksClient(host=host, token=os.getenv('DEEPCHECKS_API_TOKEN'))

You’re ready to go!#

Congratulations, you’re ready to start monitoring your models with deepchecks!

Head over to our Quickstart or see the full User Guide to get your model monitoring up and running.
